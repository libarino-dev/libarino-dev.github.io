# Página de Login

Uma tela de login moderna e responsiva para GitHub Pages.

## Características

- Design moderno e responsivo
- Validação de formulário em tempo real
- Interface intuitiva
- Compatível com dispositivos móveis

## Estrutura

- `index.html` - Estrutura HTML da página de login
- `styles.css` - Estilos CSS para a interface
- `script.js` - Lógica JavaScript para validação e interação

## Como usar

1. Faça commit e push dos arquivos para o repositório
2. O GitHub Pages irá servir automaticamente o `index.html` como página inicial
3. Acesse seu site através de `https://seu-usuario.github.io`

## Personalização

Você pode personalizar:
- Cores no arquivo `styles.css` (gradiente de fundo e botões)
- Validações no arquivo `script.js`
- Integração com sua API de autenticação no método `handleLogin()`
